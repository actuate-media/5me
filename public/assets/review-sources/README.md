# Review Source Logos (SVG)

Drop approved **SVG** logos for each review platform into this folder.

## Expected filenames
- `google.svg`
- `facebook.svg`
- `yelp.svg`
- `bbb.svg`
- `trustpilot.svg`
- `clutch.svg`
- `other.svg` (optional fallback)

## Notes
- Keep backgrounds transparent.
- Prefer single-color or official brand mark SVGs.
- If you add additional source types later, add a matching `<type>.svg` here and update the mapping in `src/lib/review-source-logos.ts`.
